Project Overview
Goal
To build a Customer Support Chatbot that can:
* Detect the user's persona (emotion, tone, and expertise level).
* Retrieve relevant knowledge base (KB) information using semantic search.
* Adapt its response tone and style to match the detected persona.
* Escalate unresolved or critical issues to human agents with full context.
